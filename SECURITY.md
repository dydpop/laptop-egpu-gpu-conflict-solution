# Security and Privacy / 安全与隐私

## 中文

`Export-GpuSupportBundle.ps1` 生成的支持包可能包含本机信息，例如：

- 操作系统版本
- BIOS 信息
- GPU、显示器和 PnP 设备状态
- 当前 Windows 图形偏好
- 电源策略摘要
- 工具日志

默认支持包会尽量脱敏，但公开上传前仍应先检查并删除你不想公开的信息。仓库的 `.gitignore` 已排除 `gpu-control/state/`、`gpu-control/support-bundles/` 和真实 `gpu-control/config/gpu-policy.json`，不要把这些内容提交到公开仓库。

只有显式使用 `-IncludePrivateDetails` 时，脚本才会导出更完整的设备、BIOS、日志、应用路径和图形偏好细节。这个参数只适合本机排查，或在脱敏后提交给可信厂商。

如果你报告问题，优先提供：

- `Check-GpuState.ps1` 的状态摘要
- WHEA 17 / TDR 是否增长
- 当前显示模式：内屏、扩展、副屏-only
- eGPU 连接路径、线材和驱动版本
- 已脱敏的支持包

## English

Support bundles generated by `Export-GpuSupportBundle.ps1` may contain local machine details, including:

- OS version
- BIOS information
- GPU, monitor, and PnP device state
- Current Windows graphics preferences
- Power policy summaries
- Toolkit logs

Review and redact support bundles before sharing them publicly. This repository ignores `gpu-control/state/` and `gpu-control/support-bundles/`; do not commit those directories to a public repository.

Keep your real `gpu-control/config/gpu-policy.json` private if it contains personal application paths, monitor IDs, or device matching patterns. The public repository tracks only `gpu-policy.example.json`.

When reporting an issue, prefer sharing:

- The state summary from `Check-GpuState.ps1`
- Whether WHEA 17 or TDR events are increasing
- Display mode: internal-only, extended, or external-only
- eGPU connection path, cable, and driver versions
- A redacted support bundle
